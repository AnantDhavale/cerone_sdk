# Data & Privacy

Cerone's free trial processes agent action data to provide runtime validation and trust scoring. By using this SDK with Cerone, you acknowledge:

- Cerone's free trial processes agent action data and limited request metadata to provide runtime validation, trust scoring, security monitoring, and abuse detection. By using this SDK with Cerone, you acknowledge:
- What we collect: agent IDs, declared purposes, capability lists, action names, parameters passed to validation, alignment scores, trust events, user agent, request timing, and hashed network identifiers derived from source IP address. We do not collect your model API keys, model outputs, or end-user data unless you explicitly include them in validation parameters.
- How we use it: to provide the validation service, maintain platform security, detect abuse, investigate misuse, improve semantic alignment models, and understand product usage. Aggregated and privacy-preserving validation patterns may be used to improve Cerone's alignment engine.
- What we do not do: we do not sell your data, share it with third parties for advertising, or use it to train models that compete with your product.
- Retention: free trial data is retained for 30 days. Paid tier data retention is configurable.
- Your model key stays yours: Cerone never sees or stores your LLM API keys. Cerone sits between your agent and its actions, not between you and your model.
- Infrastructure handling: source IP addresses may still be processed transiently by our request-handling and hosting infrastructure for rate limiting, network security, and fraud prevention, but Cerone's product telemetry stores hashed network identifiers rather than raw source IP addresses.
- For enterprise data handling, custom retention, or on-premise deployments, contact info@homersemantics.com.
